# ML Learning Lab

This repository documents my journey towards becoming an ML Engineer.

## Purpose

The goal of this repository is to build a strong foundation in Machine Learning through:

* Python
* NumPy
* Pandas
* Math for ML
* ML Fundamentals
* Scikit-learn
* Mini Projects

## Repository Structure

* `python/` – Python concepts relevant to ML
* `numpy/` – NumPy experiments and implementations
* `pandas/` – Data manipulation and analysis
* `math-for-ml/` – Mathematical concepts and implementations
* `ml-fundamentals/` – Core machine learning concepts
* `sklearn/` – Practical machine learning with Scikit-learn
* `mini-projects/` – Small projects built while learning

## Learning Philosophy

This repository focuses on:

* Understanding concepts
* Implementing ideas in code
* Building mini projects
* Documenting meaningful progress

The goal is not just to study ML, but to learn how to build with it.
